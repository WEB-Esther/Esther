※ 편집 방법 ※

  1. 자신의 컴퓨터에 있는 편집프로그램을 연다

  2. 위의 링크를 누른다

  3. View code 버튼을 누른다(파일 모양 그림)

  4. 나오는 파일들 중 수정하고 싶은 것을 찾아서 Ctrl + C

  5. 이후 Ctrl + V 로 편집프로그램에 옮긴다

  6. 편집프로그램에서 자신이 원하는 데로 바꾼다

  7. 자신의 깃허브 저장소에 다시 올린다
